The available CES secret token/PKCS#12 certificate to perform Code Pipeline actions for Rest API call. Alternatively, click Add to add a new secret text/PKCS#12 certificate using the Credentials Plugin.
Note: CES secret token can only be used for CES running on HTTP protocol and PKCS#12 certificate can only be used for CES running on HTTPS protocol. Refer to the Jenkins documentation for the Credentials Plugin.
